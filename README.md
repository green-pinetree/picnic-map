## 주요 기능

|                                                                                                                                                                                                <img src="https://github.com/green-pinetree/picnic-map/assets/81913106/61004938-da06-4b33-b82c-f14d1ca4f828" width="1000"/>                                                                                                                                                                                                | **날짜 선택 & 날씨 확인** <br> <br> <li> 날씨는 나들이를 계획하는 중요한 요소입니다. <li>사용자의 위치 기반으로 실시간 날씨 정보와 일기예보를 제공하여, 사용자들이 향후 나들이 계획을 세울 때 날씨 상태를 고려할 수 있도록 돕습니다.              |

|                                                                                                                                                                                                <img src="https://github.com/green-pinetree/picnic-map/assets/81913106/0ad3f6e2-b9df-4b04-8c25-508b15ba434d" width="500"/>                                                                                                                                                                                                | **나들이 장소 필터링** <br> <br> <li> 사용자는 필터 기능을 활용하여 공원, 두드림길, 문화시설 등을 카테고리별로 분류하여 볼 수 있습니다. <li> 이를 통해 사용자는 자신의 관심에 맞는 장소를 더욱 쉽게 찾아볼 수 있습니다.                                                                       |
| <img src="https://github.com/green-pinetree/picnic-map/assets/81913106/c14e1e0a-be99-4ff1-85a9-ea6cc2aaf1d0" width="500"/> | **나들이 장소 세부 정보** <br> <br> <li> 왼쪽 목록에서 장소를 클릭하거나, 지도 위에 아이콘으로 표시되어 있는 장소를 탭하여 자세한 정보를 확인할 수 있습니다. <li> 공원의 경우 주소, 시설, 평점, 리뷰 등을 확인할 수 있으며, 두드림길과 문화시설에 대해서도 마찬가지로 상세 정보를 제공합니다.                                                                              |
